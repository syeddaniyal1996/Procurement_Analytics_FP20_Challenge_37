# Procurement_Analytics_FP20_Challenge_37
This project was developed as part of the FP20 Analytics ZoomCharts Challenge 37, utilizing the Procurement Data to provide a comprehensive analysis of organizational spending and supplier reliability. The primary goal of this dashboard is to bring transparency to the procurement process and identify key opportunities for cost savings.

# Key Analytics & Insights
The report is structured into three primary sections:

# Spend vs Budget Analysis: 
Monitors a total spend of $979.40M against a $1.01bn budget, with IT Software (33.10%) holding the largest category share.  

# Supplier Risk & Performance: 
Categorizes suppliers according to their risk profiles (Low, Medium, High), identifying 6.49% of spend within the high-risk zone.  

# Operational Efficiency: 
Tracks critical metrics such as a 64.13% on-time delivery rate and an average lead time of 36.4 hours.  

# Compliance Tracking: 
Highlights compliance gaps by analyzing Maverick Spend (10.76%) and an Invoice Error Rate of 26.38%.  

# Technical Features
# SVG-Created KPIs: 
Utilizes custom SVG icons and indicators to provide the dashboard with a modern, professional "new look".
# ZoomCharts Integration: 
Employs interactive drill-down visuals to enable deep-dives into the data, significantly enhancing the user experience.
# Drill-through Capabilities: 
Includes advanced navigation features for regional analysis (such as Europe’s $454.94M spend) and category-wise breakdowns.
